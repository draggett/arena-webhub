ECG data from physio.net CEBS database for demo of real-time streaming.


The display shows streaming data for a pair of electrocardiogram sensors, a respiratory band and a seismocardiogram. The scenario is where someone is reported with chest pains and a health worker has attached sensors and is seeking a diagnosis from a remote cardiac specialist. The Arena Web Hub uses WebSockets for signalling events, property updates and for invoking actions and receiving the responses. The HTTP and Web Sockets connections are encrypted using transport layer security. A secure JSON Web Token is used for access control. The Web Hub automatically validates data against the data model in the thing description.
